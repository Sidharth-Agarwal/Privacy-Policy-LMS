# Privacy Policy - NEISSR App

**Last updated:** December 09, 2025

This Privacy Policy applies to the **NEISSR** mobile application built by **Niti Technologies**.

## 1. Information Collection and Use

We collect specific information required for authentication, result management, and event participation:

### A. Personal Identity & Contact Data
- **Identity:** First Name, Last Name
- **Contact:** Email Address, Phone Number
- **Profile Image:** User profile picture (optional upload)

### B. Academic Information
- Roll Number / Admission Number
- Course Enrolled (e.g., MSW, BSW)
- Current Semester/Year
- **Academic Results:** Grades, marks, and subject-wise performance data (View-only for students)

### C. App Activity Data
- **Events & Activities:** Records of events you participate in or organize within the institution

> **Note:** This App does **not** collect or process sensitive personal data such as Religion, Caste, Blood Group, Gender, Nationality, or Parental/Guardian details.

## 2. App Permissions

- **Camera & Gallery (Storage):** Required only for uploading profile pictures or event banners
- **Internet Access:** Required to fetch real-time data from secure servers

## 3. Data Storage

All data is securely stored using **Google Firebase** services:
- **Authentication:** Firebase Authentication manages login credentials
- **Database:** Cloud Firestore stores academic records with strict security rules
- **Media:** Firebase Storage hosts profile pictures and activity banners

## 4. Third-Party Services

The app uses these third-party services for analytics and crash reporting:
- [Google Play Services](https://policies.google.com/privacy)
- [Google Analytics for Firebase](https://firebase.google.com/policies/analytics)

## 5. Data Retention & Deletion

Academic data is retained while you're an active student or staff member. To delete your account or request data removal, contact the administration department or use the in-app account deletion feature.

## 6. Changes to This Privacy Policy

We may update this Privacy Policy periodically. Check this page for changes.
